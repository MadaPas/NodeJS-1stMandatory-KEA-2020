NodeJS

Node.js is an open source, cross-platform runtime environment for developing server-side and networking applications. Node.js applications are written in JavaScript, and can be run within the Node.js runtime on OS X, Microsoft Windows, and Linux.
Node.js also provides a rich library of various JavaScript modules which simplifies the development of web applications using Node.js to a great extent.

WHAT can NodeJS do?

Node.js can generate dynamic page content
Node.js can create, open, read, write, delete, and close files on the server
Node.js can collect form data
Node.js can add, delete, modify data in your database

Extra information

Node has a built tool called npm which stands for Node Package Manager. With this we can easily install modules to use for easy development. Framework modules can be found on npmjs.com.
To install a module first create a json file named package.json, then run the following command in your project’s folder:
npm install [module name]
This will generate a node_modules folder with all modules in it and it will update the package.json file where it’s possible to see all dependencies.
You can also manually add them to the package file.
After that, you will need to run npm install in your command line.
And finally, to run your project, you can simply run node [name of js file] in your command line.
Consider using Nodemon for an easier and better execution.
