Request

Request is designed to be the simplest way possible to make http calls. It supports HTTPS and follows redirects by default. Request is a module for Node handling http requests. So we use it when we need to interact with other web services like an API call.
First, we have to install the module: npm install request, and then, we import it and make a request to a web page.
